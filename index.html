jQuery Loading Plugin

Load slower!

jQuery Loading - is a simple plugin, that helps you to make your users enjoyed, while them waiting for some process ends =)

Production Version Development Version

Usage

As any jQuery plugin, Loading can be fired in one line of code:

$('#element').loading(options)

Here a simple example of simpliest loading animation.

Ofcourse, it can be configured via options, passing on creating. You can read wiki on github for more info.

How it works

It draws necessary quantity of div-block in calculated area, and then runs algorithm, that finds next dot, that must be animated, and then, gives this dot to stack of registered effects.
